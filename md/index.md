# Apuntes de Programación

Bienvenido a los apuntes de **Programación Introductoria**.

!!! info "Objetivo"
    Aprender los fundamentos para desarrollar programas desde cero.

---

## 📚 Contenido del curso

- Diseño de algoritmos
- Lenguajes de programación
- Programación orientada a objetos (Java)

---

## Cómo usar estos apuntes

1. Lee la teoría
2. Analiza los ejemplos
3. Practica con ejercicios
4. Implementa código

---

## 🛠 Requisitos

- Ninguno (nivel inicial)
- Ganas de aprender 💪

---

!!! tip "Consejo"
    No te limites a leer → escribe código mientras estudias.
